I'm a **6th-semester B.Tech Computer Science & Engineering** student at **VIT Chennai** and the **Full-Stack Developer** for the **Microsoft Innovations Club**. My core focus is **Machine Learning** and **Local AI Desktop Systems** — I build apps that run powerful models entirely on-device, with no cloud dependency.

I'm an **NVIDIA-certified developer** in RAG Agents and CUDA, and I'm currently building **[DSC](https://github.com/Ashvin-KS/DSC)** — a local AI desktop app powered by React/TypeScript, Tauri/Rust, file-system monitoring, and NVIDIA NIM models. I'm also actively contributing to open source and preparing for **GSoC 2026**.
